# Window HotEdge - 屏幕边缘触发器

基于 AutoHotkey v2.0 的屏幕边缘/区域触发工具，通过鼠标在不同区域执行滚轮/按键操作来实现快捷功能。

## 功能

### 🖱️ 基础映射
| 操作 | 功能 |
|------|------|
| Ctrl + 侧键后退 | Ctrl + 右方向键 |
| Ctrl + 侧键前进 | Ctrl + 左方向键 |
| Alt + 滚轮下 | Ctrl + 右方向键 |
| Alt + 滚轮上 | Ctrl + 左方向键 |

### 📐 屏幕边缘触发

| 区域 | 操作 | 功能 |
|------|------|------|
| 左下角 | 悬停 | 打开开始菜单 |
| 右边缘上半部 | 滚轮 | 调节音量 |
| 右边缘下半部 | 滚轮 | 切换标签页 |
| 顶部边缘左半部 | 滚轮 | 切换标签页 |
| 顶部边缘右半部 | 滚轮 | 跳转页面首/尾 |

### 📋 任务栏区域

| 区域 | 操作 | 功能 |
|------|------|------|
| 任务栏左侧 3/4 | 滚轮 | 切换虚拟桌面 |
| 任务栏左侧 3/4 | 中键 | 任务视图 |
| 任务栏右侧 1/4 | 滚轮 | 切换浏览器标签页 |

## 安装

1. 安装 [AutoHotkey v2.0](https://www.autohotkey.com/)
2. 双击运行 `window-hotedge.ahk`
3. 脚本会自动请求管理员权限

## 自定义

编辑 `.ahk` 文件修改触发区域或绑定功能。

## 要求

- Windows 10/11
- AutoHotkey v2.0+
