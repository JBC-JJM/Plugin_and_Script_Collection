# 网页目录阅读器 (TOC Reader)

自动提取网页标题（H1~H6），在页面右侧生成悬浮目录面板。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/)
2. 点击安装：[toc-reader.user.js](./toc-reader.user.js)

## 功能

- 📋 自动提取 H1~H6 标题生成目录
- 🖱️ 点击跳转（平滑滚动）
- 🔍 实时搜索过滤标题
- 🎯 滚动时自动高亮当前位置
- 🪟 面板可拖拽移动
- 💾 记住显示/隐藏状态
- 🔄 SPA 路由变化自动刷新
- 🌐 适用所有 http/https 网页
